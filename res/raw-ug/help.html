تۈرگە ئايرىش چۈشەندۈرلىشى:
-----------------
يېشىل - بۇ ئەپنىڭ يىشىش ئۇسۇللىرى بەك كۆپ؛
سېرىق /sdcard/LuckyPatcher/ دىن بۇ ئەپىنڭ-
قېنىق كۆك - بۇ ئەپتە گۇگىل ئېلانى بايقالدى؛
ھال رەڭ - بۇ ئەپ قوزغىتىش تىزىملىكىگە قوشۇلدى؛
قىزىل - بۇ ئەپنى يەشكىلى بولمايدۇ،چۈنكى بۇ ئەپ ئىككى قىسىمدىن تۈزۈلگەن(ئەپ ھۆججىتى ۋە كەسپى مەخپى نۇمۇر).بۇ ئەھۋالدا،ئىككى قىسىمنى بىرلا ۋاقىتتا ياكى خالىغان بىرسىنى يىشش كىرەك؛
قىزغۇچ سېرىق - بۇ بولسا سېستىما ئەپى،سېستىما ئەپىگە مەشخۇلات قىلغاندا چوقۇم ئېھتىياتچان بۇلۇڭ،بۇ بەلكىم ئەپ ياكى سېستىما ئۈچۈن ناچار ئاقىۋەت ئېلىپ كىلىشى مۇمكىن؛
قوزغىتىش تىزىملىكى - يانفۇننى قوزغاتقاندىن كىيىن ئاپتۇماتىك يىشلدىغان ئەپنىڭ تىزىملىكى.بۇ تىزىملىك پەقەتلا ئەپ "سو."ھۆججىتىنى لازىم بولغاندا ئىشلىتىلدۇ.لىكىن ئۆزلىكىدىن يىشلدىغان يەشمە يوق،پەقەت ئەپنىڭ كونا نەشىرىنىڭ زاپاسىمىسىنى قايتا ئىشلەتمىگەندە؛

تۆت يۇپۇرماق -ئەپنىڭ يىشىش نۇسخىسىنى ئۆزگەرتىلگەندىن كىيىنكى ھالىتىنى كۆرسىتىدۇ؛
يۇلتۇز - ئەپنىڭ مەۋھۈم ئۈسكىنە بوففېر ساقلانمىنىڭ ئۆزگەرتىلگەنلىكىنى بىلدۈردۇ،ئۇنداق بولمايدىكەن ئەپنىڭ ھازىرقى ھالىتىنى ساقلاپ قالغانلىقىنى بىلدۈردۇ،ئەپنى يەنە يېڭىلغانغا ياكى يۇيۋەتكەنگە قەدەر داۋاملىشىدۇ.ئەگەر ODEX نى " ODEX ئۆزگەرتشىنى چىقىرۋېتىش"ئارقىلىق چىقىرۋەتكەندە،ئەپ يىششتىن بۇرۇنقى ھالىتىگە قايتىدۇ.ئەپنى يېڭىلغاندىن كىيىن،مۇنداق مەسلە كىلىپ چىقىدۇ:ئەپ بازىرى بوشلۇق يىتىشمىگەن دەپ ئەسكەرتىدۇ،بۇ ۋاقىتتا تەلەي يامىقىنى قوزغىتىپ ھەم تاقاپ،ئەپنى قاچىلاپ سىناپ بېقىڭ.ئەگەر بۇنداق خاتالىق يۇيۋىتىلگەن ئەپتە كۆرۈلسە،"كاشلىنى رەت بۇيچە تەكشۈرۈش->زاپاس ۋە تۈزەشىنى چىقىرۋېتش"نى تاللاڭ-تۈزتىلگەن بارلىق ئەپنىڭ مەۋھۈم ئۈسكىنىدىكى بوففېر ساقلانما يۇيلىپ كىتىدۇ!>

Instructions:
--------------
For Remove License Verification:
- Use as target only original apk files signed by the developer;
- Tap to Software and select "Remove License Verification!";
- Select "Auto mode" and test Application with Internet;
- If the result is negative, Select "Auto mode!(Inverse)" or "Extreme Auto mode";
- If you're lucky, your program will be registered;
- To return Application to its original state, you need to tap "Remove ODEX with Changes" or "Restore".

For Remove Google Ads:
- Tap to Software and select "Remove Google Ads!";
- Run and test Application with Internet;
- If you're lucky, Google Ads disappeared;
- To return Application to its original state, you need to tap "Remove ODEX with Changes" or "Restore".

For Custom Patch:
- Tap to Software and select "Custom Patch!";
- Run and test Application with Internet;
- If you're lucky, patch work;
- To return Application to its original state, you need to tap "Remove ODEX with Changes" or "Restore".

For Change Permissions:
- Tap to Software and select "Change Permissions";
- Tap to Permission for disable (red);
- Tap to Permission for enable (green);
- You must tap Apply (Reboot).

For Create Modified APK:
- Tap to Software and select "Create Modified APK";
- Select Patch for Modification of APK;
- Modified installation file will be in folder /sdcard/LuckPatcher/Modified/ ;
- You should know that a rebuild apk-file is not the same as the patch is already installed application. Since the as opposed to simply patch the application is very easy to test yourself to changes, and thus work correctly so faly The install will not.

PS: Patcher closes the application, when makes patches. This is necessary so that the new changes earned. But sometimes you may need to restart the device itself or use a third-party tools to kill the process of Applications.
