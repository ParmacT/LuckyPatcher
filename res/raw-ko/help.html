사용하시려면 반드시 루트 권한이 필요합니다!

설명:
-----------------
녹색 - 이 프로그램은 라이센스가 발견되었음을 의미합니다.
노란색 - 해당 패치는 /sdcard/LuckyPatcher/ 폴더에서 사용자 패치가 발견되었음을 의미합니다.
청록색 - 구글 광고가 발견되었음을 의미합니다.
자홍색 - 해당 프로그램이 부팅리스트에 있음을 의미합니다.
불가능 - 해당 프로그램은 라이센스를 제거할 수 없음을 의미합니다. 이러한 표시가 나타나는 두가지 경우가 있습니다. (애플리케이션 그리고 Apk 프로키). 이러한 경우에는 양쪽 모두 패치를 하거나 둘 중 한개를 패치해주시면 됩니다.
주황색 - 시스템 애플리케이션을 의미합니다. 주황색의 어플들은 어떠한 작업을 하든 신중히 하셔야 하며, 잘못하면 시스템 작동에 문제를 줄 수 있습니다.
부팅리스트 - 장치 부팅시 패치를 적용할 프로그램들의 목록입니다. 이 기능은 ".so" 라이브러리의 부팅마다 다시 수정되도록 만들어진 어플에 사용합니다. 하지만 사용자 패이는 존재하지 않으며, 예를들어 구 티타늄 백업이 있습니다.

Star - dalvik-cache 어플리케이션이 수정되었음을 의미하며 현재의 상태를 게속 유지할것입니다. 어플리케이션을 업데이트할때, 충분한 공간이 있지 않다는 알림이 뜨는 문제가 발생한다면, \"dalvik-cache 변화 수정-> 업데이트를 위해 수정값 삭제 \"를 선택하세요 그리고 다시 어플리케이션 설치를 시도하세요, 만약 그래도 오류가 발생한다면 오류해결->업데이트를 위해 수정,백업값 삭제\"를 클릭하세요 - 이것은 모든 어플의 dalvik-cache를 제거합니다

메뉴 설명:
-----------------
라이센스 확인 제거 (자동 모드!) - 라이센스를 제거하는 기능입니다. 최대한 애플리케이션 구동에 문제가 생기지 않도록 라이센스를 제거하는 방식입니다.
라이센스 확인 제거 (고급 자동 모드!) - 라이센스 제거 기능입니다. 모든 방법을 사용하며, 애플리케이션에 문제가 발생할 수 있습니다. 대부분의 오류는 오프라인 문제입니다.
라이센스 확인 제거 (수동) - 라이센스를 제거하는 기능입니다. 사용자가 수동으로 라이센스 제거를 하게됩니다.
라이센스 확인 제거 (패턴 선택) - 패치를 적용할 패턴을 사용자가 직접 선택할 수 있습니다.
구글 광고 제거! - 구글 광고 제거를 시도하는 기능입니다. 특별한 경우에는 애플리케이션의 광고는 제거되지만 해당 광고 태그(발견할 수 없는 상태!)가 제거되지 않는 경우가 발생합니다.
변경된 값의 ODEX 제거 - 남아있는 변경값의 ODEX파일을 제거하는 기능입니다. 이 기능을 통해 애플리케이션을 원본 상태로 되돌릴 수 있습니다.
퍼미션 변경 - 애플리케이션의 퍼미션을 제거합니다. 이 기능을 사용하실때는 조심하셔야 합니다. 일부 기기에선, 이 방법이 무한 재부팅 현상을 일으킬 수 있습니다. 변경된 퍼미션 값을 적용하기 전에 반드시 시스템을 백업해두세요.
수정된 Apk파일 생성 - 패치를 적용한 새로운 설치 패키지 apk파일을 생성하는 기능입니다. 반드시 알아두셔야 할점은 이미 설치된 애플리케이션과 패치가 적용된 애플리케이션은 같지 않다는 점을 명심하셔야 합니다. 이 기능의 장점은 변경값을 스스로 확인해보실 수 있으며, 다시 패치를 하지 않고 간단하게 설치만으로 어플리케이션을 정상적으로 사용할 수 있다는 점입니다.
재부팅시 패치 적용 - 장치 부팅시 패치를 적용할 프로그램들의 목록입니다. 이 기능은 ".so" 라이브러리의 부팅마다 다시 수정되도록 만들어진 어플에 사용합니다. 하지만 사용자 패이는 존재하지 않으며, 예를들어 구 티타늄 백업이 있습니다.

사용법:
--------------
라이센스 확인을 제거하려면:
- 반드시 개발자의 원본 apk 파일을 통해 패치를 진행하셔야 합니다.
- 소프트웨어를 클릭하시고 "라이센스 확인 제거!"를 선택하세요.
- "자동 모드"를 선택하시고 인터넷이 연결된 상태에서 애플리케이션을 테스트해보세요.
- 만약 결과가 실패라면, "고급 자동 모드"를 선택하세요.
- 만약 운이 좋다면, 해당 프로그램은 라이센스가 제거되어있을 겁니다.
- 애플리케이션의 원본 상태로 되돌리시려면, "변경된 값의 ODEX 제거"기능을 사용하시면 됩니다.

구글 광고를 제거하려면:
- 소프트웨어를 클릭하시고 "구글 광고 제거!"를 선택하세요.
- 실행하셔서 인터넷이 연결된 상태에서 테스트 해보세요.
- 만약 운이 좋다면, 해당 프로그램의 구글 광고가 더이상 나타나지 않을겁니다.
- 애플리케이션의 원본 상태로 되돌리시려면, "변경된 값의 ODEX 제거"기능을 사용하시면 됩니다.

사용자 패치를 하려면:
- 소프트웨어를 클릭하시고 "사용자 패치!"를 선택하세요.
- 실행하셔서 인터넷이 연결된 상태에서 테스트 해보세요.
- 만약 운이 좋다면, 패치가 정상적으로 적용되었을겁니다.
- 애플리케이션의 원본 상태로 되돌리시려면, "변경된 값의 ODEX 제거"기능을 사용하시면 됩니다.

퍼미션을 변경하려면:
- 소프트웨어를 클릭하시고 "퍼미션 변경"를 선택하세요.
- 비활성화하려면 퍼미션을 클릭하세요 (빨간색).
- 활성화하려면 퍼미션을 클릭하세요 (녹색).
- 반드시 적용 (재부팅)을 누르셔야 합니다.

수정된 APK 파일을 생성하려면:
- 소프트웨어를 클릭하시고 "수정된 Apk파일 생성"을 선택하세요.
- Apk에서 적용할 패치를 선택하세요.
- 수정된 설치 파일은 /sdcard/LuckPatcher/Modified/ 폴더에 저장됩니다 .
- 이미 설치된 애플리케이션과 수정되어 생성된 Apk파일은 같은 파일이 아님을 반드시 알아두세요. 이 기능의 장점은 변경값을 스스로 확인해보실 수 있으며, 다시 패치를 하지 않고 간단하게 설치만으로 어플리케이션을 정상적으로 사용할 수 있다는 점입니다.

PS: 패치파일을 생성할때 패쳐는 애플리케이션을 종료시킵니다. 새로운 변경값을 얻기위해서 반드시 필요한 작업이며, 특별한 경우에는 장치의 재부팅이 필요하거나 애플리케이션의 작업을 종료하기위해 다른 도구를 사용해야하는 경우도 있습니다.
