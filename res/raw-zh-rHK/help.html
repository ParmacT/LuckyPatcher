分類說明：
---------------------------------
綠色 - 此應用程式有很大機會被破解；
黃色 - 可在/sdcard/LuckyPatcher/找到自定義破解；
青色 - 發現該應用程式有谷歌廣告；
洋紅 - 此應用程式已添加至啟動列表；
紫色 - 支持谷歌內購的應用程式；
紅色 - 此應用程式沒有機會被破解，該異常是因為應用程式擁有兩部分（應用程式和專業版密鑰）。在這種情況下，需要同時破解兩個或其中一個應用程式；
橙色 - 此為系統應用程式，對系統應用程式執行任何操作時，請務必謹慎，它可能對應用程式或作業系統造成嚴重的不良後果。
啟動列表 - 開機後自動應用破解的應用程式列表。此列表僅在應用程式需要修復庫檔案 「.so」 時才使用。但不存在自定義破解，除非用於已不再使用的舊版本鈦備份。

四葉草 - 顯示應用程式破解模式更改後的狀態；
星標 - 意味著應用程式的虛擬機快取已被修復（已創建 ODEX 更改），相反則意味著程式將保持當前的狀態，直至應用程式被再次更新或移除。如果此 ODEX 使用 「移除 ODEX 更改」 來移除，那應用程式將會完全恢復到被應用破解前的狀態。當您更新應用程式時，問題就會出現：電子市場將會提示空間不足，此時，運行幸運破解器並關閉，再嘗試安裝該應用程式。如果這樣的錯誤出現在已卸載的應用程式中，請選擇 「故障排除 -> 移除修復和備份（更新程式）」 - 所有修復的應用程式虛擬機快取將會被移除！

上下文菜單：
---------------------------------
移除授權驗証（自動模式）
- 嘗試移除應用程式的授權驗証。將會使用小部分的破解模組來進行破解以確保應用程式的最大穩定性。
移除授權驗証（自動模式(逆向破解)）
- 此模式是 「自動模式」 的逆向破解模式。如果 「自動模式」 破解後沒有效果，可以嘗試使用此項進行破解。
移除授權驗証（極限自動模式）
- 嘗試移除應用程式的授權驗証。將會使用所有的破解模組來進行破解，但破解後的應用程式不穩定，在使用過程中會產生更多的錯誤。
移除授權驗証（手動）
- 嘗試移除應用程式的授權驗証。將會手動選擇破解模式來進行強力移除應用程式的授權驗証。
移除授權驗証（選擇破解模式）
- 您需要選擇一組破解模組來對應用程式進行破解。
移除谷歌廣告！
- 嘗試移除應用程式自帶的谷歌廣告。有時候應用程式會處於未發現任何情況的狀態，您需要從未找到廣告標籤的應用程式中進行移除谷歌廣告。
移除 ODEX 更改
- 移除 ODEX 檔案及包含其更改，將應用程式恢復到初始狀態。
更改許可權和活動項
- 移除應用程式請求系統的許可權。請謹慎對此項進行操作，由於在某些設備中更改許可權後會導致無限重啟或其他不良後果，因此，在您嘗試更改許可權前建議先對系統進行備份。
創建修改的 APK 版本
- 基於可用的破解來創建一個 APK 安裝包。請注意：重建的 APK 檔案對於已安裝的程式被應用破解的效果並不同，由於應用破解的程式可以很簡易的測試自身的更改並使用正確，但安裝重建的 APK 檔案使用效果並非一樣。
重啟時應用破解
- 此項僅在應用程式需要修復庫檔案 「.so」 時才使用，但不存在自定義破解，除非用於已不再使用的舊版本鈦備份。如果應用程式在重啟後應用了破解出現不穩定，它就需要添加不超過一個自定義破解。如果是應用程式使用時必需的，它將會被寫入一個自定義破解。

手動破解器！
- 調試應用程式的破解器。
備份！
- 備份 APK 檔案到 /sdcard/LuckyPatcher/Backup/ 資料夾。
恢復！
- 從 /sdcard/LuckyPatcher/Backup/ 資料夾恢復備份的 APK 檔案。


複製更改到虛擬機快取！
- 在預設模式下的所有更改將會保留 ODEX 檔案。但某些 ODEX 檔案卻不能工作於相應的應用程式（例如：當移動應用程式到外部 SD 卡時），因此，複製更改到虛擬機快取就可以解決此問題。但是，如果您只想重新安裝該應用程式，您可以直接恢復它。
備份重新安裝的 APK 檔案！
- 複製 APK 檔案到 SD 卡的 「LuckyPatcher」 數據資料夾。複製成功後可以從這個 APK 檔案重新安裝（恢復）相應的應用程式。
移除虛擬機快取！
- 移除此應用程式在虛擬機快取創建後的 ODEX 檔案以釋放空間。如果 ODEX 檔案不能工作於該應用程式，請重新安裝此應用程式。

破解說明：
---------------------------------
移除授權驗証：
- 只能用於開發者簽名的原始 APK 檔案！
- 長按應用程式並選擇 「移除授權驗証」；
- 選擇 「自動模式」 並連接網路測試應用程式；
- 若破解沒有效果，再選擇 「極限自動模式」；
- 如果您夠幸運，應用程式將會被破解；
- 點擊 「移除 ODEX 更改」 可恢復應用程式到初始狀態。

移除谷歌廣告：
- 長按應用程式並選擇 「移除谷歌廣告」；
- 運行並連接網路測試應用程式；
- 如果您夠幸運，谷歌廣告將會被移除；
- 點擊 「移除 ODEX 更改」 可恢復應用程式到初始狀態。

自定義破解：
- 長按應用程式並選擇 「自定義破解」；
- 運行並連接網路測試應用程式；
- 如果您夠幸運，應用程式將會被破解；
- 點擊 「移除 ODEX 更改」 可恢復應用程式到初始狀態。

更改許可權和活動項：
- 長按應用程式並選擇 「更改許可權和活動項」；
- 單擊不要的許可權可停用 （紅色）；
- 單擊需要的許可權可啟用 （綠色）；
- 最後必須點擊 「立即重啟應用更改」。

創建修改的 APK 版本：
- 長按應用程式並選擇 「創建修改的 APK 版本」；
- 再根據需要選擇要創建的 APK 版本；
- 已修改的安裝檔案將會保存在 /sdcard/LuckPatcher/Modified/ 資料夾；
- 請注意：重建的 APK 檔案對於已安裝的程式被應用破解的效果並不同，由於應用破解的程式可以很簡易的測試自身的更改並使用正確，但安裝重建的 APK 檔案使用效果並非一樣。

備註：當應用破解時，幸運破解器會關閉其應用程式，為了獲取新的更改必需如此。但有時您可能需要重新啟動設備或使用第三方工具來終止應用程式的進程。
