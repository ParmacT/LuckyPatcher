Aplikasi ini membutuhkan Root akses!

Penjelasan singkat:
-----------------
Hijau – Peluang aplikasi ter-register sangat besar;
Kuning – Aplikasi ini mempunyai custom patch yang ada pada folder /sdcard/LuckyPatcher/;
Cyan – Aplikasi ini memuat iklan google;
Magenta – Program ini termasuk dalam daftar boot;
Merah – Program ini tidak bisa diregister-kan. Pengecualian untuk aplikasi yang memiliki 2 file terpisah (Aplikasi dan APK-Pro-Key). Dalam beberapa kasus bsa dilakukan patch pada salah satunya.;
Orange – Sistem Aplikasi. Berhati-hatilah dalam melakukan perubahan pada aplikasi sistem karena dapat menyebabkan sistem error/tidak stabil.;
Daftar Boot – Sebuah daftar aplikasi yang dipatch saat di boot. Hanya aplikasi yang membutuhkan perbaikan library ".so" dan tidak mempunyai custom patch seperti halnya Titanium Backup versi lawas

Bintang - dalvik-cache aplikasi telah dirubah (ODEX with Changes dibuat), hal ini juga berarti aplikasi tersebut tetap pada keadaan itu sampai ada update, atau penghapusan aplikasi. Jika ODEX dihilangkan dengan "Hapus ODEX with Changes", maka aplikasi tersebut akan kembali seperti sebelum di patch. Jika aplikasi tersebut di update, maka akan muncul masalah kalau google market kekurangan memori penyimpanan, alternatifnya jalankan Lucky patcher kemudian tutup aplikasinya serta instal ulang aplikasi tersebut, jika masalah muncul untuk aplikasi yang sudah dihapus maka jalankan Lucky patcher "Masalah->hapus perbaikan dan cadangan (Update app)" – semua perbaikan dalvik-cache dari aplikasi yang ada akan dihilangkan.

Menu Kontek:
-----------------
Hapus Verifikasi Lisensi (Auto mode!) – Coba hapus verifikasi lisensi. Menggunakan sebagian kecil pola untuk menjaga kestabilan aplikasi.
Hapus Verifikasi Lisensi (Extreme Auto mode!) - Coba hapus verifikasi lisensi. Semua pola dipakai. Aplikasi tidak akan stabil. Masalah offline sering muncul.
Hapus Verifikasi Lisensi (Manual) - Coba hapus verifikasi lisensi. Hapus lisensi dengan metode brute force.
Hapus Verifikasi Lisensi (Selected Patterns) – Anda dapat memilih pola yang dipakai.
Hapus Iklan Google! – Coba hapus iklan google. Kadangkala akan menghapus iklan lainnya dari aplikasi yang mempunyai label “tidak ada iklan”.
Hapus ODEX with Changes - Hapus ODEX yang mengalami perubahan. Yang akan mengembalikan status aplikasi seperti sedia kala. Ubah Per-Ijinan – Hapus perijinan aplikasi.berhati-hatilah saat melakukannyakarena metode ini dapat menyebabkan gadget anda restat berulang-ulang ataupun konsekuensi lainnya. Selalu buat cadangan data sebelum melakukan perubahan perijinan. Buat APK Termodif - Buat aplikasi baru dengan perubahan dan patch. Anda harus tahu kalau aplikasi baru ini tidaklah sama persis seperti aplikasi sebelum dipatch. Dikarenakan aplikasi ini telah dibuat lengkap dengan patchnya, yang kemungkinan akan bekerja dengan baik ataupun juga tidak bisa diinstal sama sekali.
Patch saat Reboot – hanya aplikasi yang membutuhkan perbaikan library ".so". tidak adanya custom patch, kecuali untuk titanium backup versi lawas, yang sudah tidak digunakan lagi. Sangat tidak stabil, dan diharapkan untuk tidak menambahan lebih dari satu patch. Jika sangat dibutuhkan sekali maka akan dibuatkan custom patchnya.

Penjelasan:
--------------
Untuk Hapus Verifikasi Lisensi:
- Selalu gunakan file apliksai yang telah di-sign oleh developer
- Tuju aplikasi dan pilih " Hapus Verifikasi Lisensi!";
- Pilih "Auto mode" lalu uji aplikasi dengan internet;
- Jika hasilnya negatif, Pilih "Extreme Auto mode";
- Jika beruntung, maka program tersebut akan diregisterkan;
- Untuk mengembalikan aplikasi ke kondisi awal, tinggal pilih "Hapus ODEX with Changes".

Untuk Hapus Iklan Google!:
- Tuju aplikasi dan pilih " Hapus Iklan Google!";
- Jalankan dan uji aplikasi dengan internet;
- Jika beruntung, Iklan Google akan hilang;
- Untuk mengembalikan aplikasi ke kondisi awal, tinggal pilih "Hapus ODEX with Changes".

Untuk Custom Patch:
- Tuju aplikasi dan pilih " Custom Patch!";
- Jalankan dan uji aplikasi dengan internet;
- Jika beruntung, patch akan bekerja
- Untuk mengembalikan aplikasi ke kondisi awal, tinggal pilih "Hapus ODEX with Changes".

Untuk Ubah Per-Ijinan:
- Tuju aplikasi dan pilih " Ubah Per-Izinan ";
- pilih perijinan untuk membatalkan (merah);
- pilih perijinan untuk menyetujui (hijau);
- Anda harus pilih ‘Terapkan’ (Reboot).

Untuk Buat APK Termodifikasi:
- Tuju aplikasi dan pilih " Buat APK Termodif";
- Pilih patch untuk memodifikasi APK;
- hasil modif terdapat pada folder /sdcard/LuckPatcher/Modified/ ;
- Anda harus tahu kalau aplikasi baru ini tidaklah sama persis seperti aplikasi sebelum dipatch. Dikarenakan aplikasi ini telah dibuat lengkap dengan patchnya, yang kemungkinan akan bekerja dengan baik ataupun juga tidak bisa diinstal sama sekali.   PS: patcher akan menutup aplikasi saat membuat patches. Hal Ini sangat dibutuhkan untuk hasil yang maksimal. Tapi kadangkala anda perlu reboot untuk menerapkan perubahan atau gunakan aplikasi pihak ketiga untuk menutup aplikasi tersebut
